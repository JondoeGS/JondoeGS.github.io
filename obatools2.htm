How to use this table:
The minimum and the maximum of each stat is governed by two similar formula

MinValue= (MinX + ((MaxX - MinX) / 2 * Rarity)) * Level^2 + MinS + ((MaxS - MinS) / 2 * Rarity)

MaxValue= (MaxX + ((MaxX - MinX) / 2 * Rarity)) * Level^2 + MaxS + ((MaxS - MinS) / 2 * Rarity)

Rarity- Basic: 0, Common: 1, Uncommon: 2, Epic: 3, Legendary: 4, Rogue: 5, Mythical: 6, Ancient: 7, Galactic: 8

Change the MinX, MaxX, MinS, and MaxS to adjust the variables in these formula. Experiment with different levels and rarity.
